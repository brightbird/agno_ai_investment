## 版本号规范

本项目遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)：

- **主版本号 (MAJOR)**: 当您做了不兼容的 API 修改
- **次版本号 (MINOR)**: 当您做了向下兼容的功能性新增
- **修订号 (PATCH)**: 当您做了向下兼容的 Bug 修复
