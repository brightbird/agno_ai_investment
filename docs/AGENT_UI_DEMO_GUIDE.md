# 🎯 Agno AI 投资分析系统 - 统一Playground演示指南

## 📋 概述

本指南将详细介绍如何在 **统一的Playground界面** 中同时体验个人投资大师Agents和投资团队Teams的功能。这是Agno AI投资分析系统的核心特色 - 在同一个界面中无缝切换不同的分析模式。

## ✨ 核心优势

- 🔄 **无缝切换**: 在同一界面中体验个人专家和团队协作
- 🎯 **双模式分析**: Agents模式 + Teams模式
- 🌐 **统一体验**: 一个Playground，两种分析方式
- 📊 **对比分析**: 直接对比个人观点和团队综合建议

## 🚀 快速开始

### 1. 启动统一Playground

```bash
# 进入项目目录
cd agno_ai_investment

# 启动统一Playground服务
python apps/playground.py

# 服务启动后会显示：
# ✅ Playground 初始化完成
# 💡 可用的投资大师: [9个个人Agents]
# 🏆 可用的投资团队: [1个投资团队]
```

### 2. 访问Web界面

1. 打开浏览器，访问：**https://app.agno.com/playground**
2. 在端点配置中添加：`localhost:7777`
3. 连接成功后，您将看到两个主要部分：
   - **Agents**: 个人投资大师列表
   - **Teams**: 投资团队列表

- **特色**: 价值投资 + 多学科思维的完美结合

## 📊 双模式对比演示

### 建议的对比演示流程

1. **同一股票，不同模式分析**:
   - 先用个人Agent（如Warren Buffett）分析AAPL
   - 再用投资团队分析同一只股票
   - 对比分析深度和角度差异

2. **不同复杂度问题测试**:
   - 简单问题：个人Agent可能更直接
   - 复杂问题：投资团队提供更全面的分析

### 对比要点

| 维度 | 个人Agents | 投资团队 |
|------|------------|----------|
| **分析深度** | 专业领域深度分析 | 多角度综合分析 |
| **响应速度** | 较快 | 稍慢（需要协调） |
| **观点多样性** | 单一专家视角 | 多专家协作视角 |
| **适用场景** | 特定风格需求 | 全面分析需求 |
| **报告结构** | 专家个人风格 | 标准化结构报告 |

## 🔧 演示技巧

### 最佳实践

1. **循序渐进**: 先演示个人Agent，再展示团队功能
2. **对比展示**: 用同一个问题测试不同模式
3. **突出特色**: 强调统一界面的便利性
4. **实时数据**: 展示股票数据的实时获取能力

### 常见问题处理

1. **连接问题**: 确保端点配置正确
2. **响应慢**: 团队分析需要更多时间，属正常现象
3. **API限制**: 如遇到API限制，可以展示历史对话记录
